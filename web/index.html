🍪 如何获取 cookies.txt

          
适用于 YouTube、Bilibili、X / Twitter、Instagram 等需要登录或验证的网站。

        

        
      

      

        

          先确认：你已经能在浏览器里正常打开这个页面或视频，并且使用的是正确的账号与浏览器配置文件。
        

        
    
          
  1. 在你平时能正常打开目标内容的浏览器里登录对应网站。
    2. 
          
  2. 使用支持导出 cookies.txt（Netscape / Mozilla 格式）的浏览器工具或扩展导出当前站点的 cookies。Chrome 推荐使用 Get cookies.txt LOCALLY。
    3. 
          
  3. 保存为 cookies.txt，回到本工具点击“选择 cookies.txt”导入。
    4. 
          
  4. 如果网站经常风控或导出后仍失效，请重新导出一份新的 cookies.txt。
    5. 
        

        

          YouTube 额外建议
          
优先在无痕 / 隐私窗口中登录 YouTube，确认视频可播放后，在同一窗口访问 https://www.youtube.com/robots.txt 再导出 cookies.txt，导出后关闭该无痕窗口。

        

        

          注意事项
          
    
            
  • cookies.txt 含有登录信息，请勿随意发送给他人。
    • 
            
  • 必须从你实际登录目标网站的同一个浏览器、同一个配置文件导出。
    • 
            
  • 不同网站的 cookie 过期速度不同，失效后重新导出即可。
